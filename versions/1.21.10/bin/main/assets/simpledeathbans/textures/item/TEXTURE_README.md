# Texture Placeholder

This folder should contain:
- `resurrection_totem.png` - Resurrection Totem texture
- `soul_link_totem.png` - Soul Link Totem texture

Each texture should be a 16x16 pixel PNG file.

## Resurrection Totem
Suggested design:
- A glowing golden totem with purple/dark accents
- Similar style to the vanilla Totem of Undying but with darker, more ominous colors
- Consider adding small soul-fire effects or netherite-colored borders

## Soul Link Totem
Suggested design:
- Two intertwined souls or heart shapes
- Purple/amethyst and teal/ender pearl color scheme
- Glowing ethereal appearance
- Could show two connected spirits/wisps

You can create these textures using:
- Aseprite
- GIMP
- Piskel (free online pixel art editor)
- Or any other image editor

Place the completed PNG files in this directory:
`src/main/resources/assets/simpledeathbans/textures/item/`
